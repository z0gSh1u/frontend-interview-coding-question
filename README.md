# 前端面试编码题

本项目收录了一系列前端开发岗位面试的编码题，主要囊括三大部分：

- 数据结构与算法
- JS语言应用
- CSS页面布局

为一系列面试高频题提供了代码实现，避免你只是看着面经在脑子里空想。

## 开始

- Clone本项目
- 执行`npm install`安装
- 执行`npm run serve`，访问`http://localhost:4001`即可

建议你充分结合审查元素功能（观察Console、观察CSS布局），同时结合具体代码来学习具体实现。大部分代码都包含了简明扼要的注释和实现关键说明。
